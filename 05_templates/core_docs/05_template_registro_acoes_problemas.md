## 1. Introdução & Objetivo

Este documento serve como registro central para acompanhamento de Ações e Problemas relacionados ao **Programa de Transformação DEI**.

- **Problemas:** Obstáculos ou preocupações que surgiram e que requerem resolução para não impactar o cronograma, custo ou qualidade do projeto.
- **Ações:** Tarefas específicas que precisam ser concluídas para avançar o projeto ou resolver um problema.

O objetivo deste registro é garantir que todos os itens sejam capturados, atribuídos e resolvidos de forma transparente, servindo como base para as reuniões de status.

## 2. Definições do Registro

### 2.1 Definições de Status

- **Aberto:** O item foi identificado, mas o trabalho ainda não começou.
- **Em Andamento:** O trabalho está sendo realizado para tratar o item.
- **Bloqueado:** O progresso está parado devido a uma dependência.
- **Fechado:** O item foi resolvido/concluído e verificado.

### 2.2 Definições de Prioridade

- **Alta:** Deve ser tratada com urgência; impacto significativo no projeto.
- **Média:** Importante, deve ser tratada em tempo hábil.
- **Baixa:** Desejável tratar, mas impacto mínimo se atrasar.
